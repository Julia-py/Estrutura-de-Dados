# 📚 Estrutura de Dados — Implementação de Listas Encadeadas em Java

Este projeto contém a implementação de diferentes tipos de listas encadeadas em Java, desenvolvidas com fins acadêmicos na disciplina de Estrutura de Dados.

---

## 🚀 Tipos de Listas Implementadas

✔️ Lista Encadeada Simples
✔️ Lista Duplamente Encadeada
✔️ Lista Circular Simples
✔️ Lista Duplamente Encadeada Circular

---

## 🧠 Conceitos Aplicados

* Manipulação de ponteiros (referências)
* Alocação dinâmica de memória
* Estruturas lineares
* Encapsulamento em Java
* Operações básicas:

  * Inserção
  * Remoção
  * Busca (`get`)
  * Tamanho da lista (`size`)

---

📁 projeto

---

## ⚙️ Funcionalidades

Cada estrutura implementa os seguintes métodos:

* `add(index, elemento)` → adiciona elemento na posição
* `add(elemento)` → adiciona no final
* `remove(index)` → remove elemento da posição
* `get(index)` → retorna o nodo na posição
* `size()` → retorna o tamanho da lista

---

## 🔍 Exemplos de Uso

```java
Lista lista = new Lista();

lista.add(10);
lista.add(20);
lista.add(1, 15);

lista.remove(0);

System.out.println(lista.get(1));
```

## 🎯 Objetivo

O objetivo deste projeto é consolidar o aprendizado sobre listas encadeadas, compreendendo na prática:

* Diferença entre listas simples e duplas
* Funcionamento de listas circulares
* Manipulação correta de referências
* Tratamento de casos especiais (início, meio e fim)
